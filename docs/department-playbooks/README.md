# Department Playbooks

Organization Bible が定義する各部門の、日々の業務運用を具体化したドキュメント群。
Development Bible 第30章「Department Playbooks(営業・開発・経理など部門別運用書)」
に対応する。

## 一覧

- [sales.md](sales.md) — AI営業本部 営業部(Epic β-001 で最初に実運用投入)
- [development.md](development.md) — AI開発本部
- [accounting.md](accounting.md) — AI管理本部 経理部門(サンプル)

他部門は必要になった時点で本テンプレートに従い追加する。

## テンプレート

新しい部門のPlaybookを追加する場合、以下の見出しを最低限含める。

```markdown
# <本部名> <部門名> Playbook

## 役割
## 日次業務フロー
## 使用ツール・連携先
## KPI
## エスカレーション・承認ルート
## 禁止事項(部門固有)
```
