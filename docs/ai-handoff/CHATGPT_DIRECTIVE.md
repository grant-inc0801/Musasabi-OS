# ChatGPT Directive

## Directive ID
D-20260706-006

## Title
β版UI追加修正：アバターサイズ調整・左サイドバー総合指標・部門進捗表示

## 決定
前回Directiveの修正に加えて、以下を追加実装してください。

1. アバタークリック時のミニパネルで、アバターサイズを調整できるようにする。
2. 左サイドバーに総合計のAI社員数と売上を表示する。
3. 左サイドバーまたは部門一覧内に、各部門のタスク進捗と作業内容の要約を表示する。

## 背景
Musasabi OSは「部門ごとにAI社員が稼働するAI会社OS」です。

そのため、左サイドバーは単なる部門メニューではなく、会社全体と各部門の状態を一目で確認できるミニ経営ダッシュボードとして機能させてください。

## 追加UI修正指示

### 1. アバターサイズ調整

アバタークリック時に表示されるミニパネル内に、アバターサイズ調整UIを追加してください。

要件：

- 小 / 中 / 大 のプリセットを用意する
- 可能であればスライダーでも調整できるようにする
- 設定値を保持する
- アバターのみ表示状態にもサイズ設定を反映する
- ミニパネル表示中もレイアウトが崩れないようにする

初期値は「中」とします。

### 2. 左サイドバーの総合指標

左サイドバー上部または下部に、会社全体のサマリーを表示してください。

表示項目：

- 総AI社員数
- 総売上
- 本日タスク進捗
- 稼働中部門数

β版ではMock値で構いません。

例：

```text
全体サマリー
AI社員：12名
総売上：¥1,250,000
本日進捗：68%
稼働部門：3部門
```

### 3. 部門ごとの進捗・作業内容表示

左サイドバーの各部門表示、または部門一覧カードに以下を表示してください。

- 部門名
- 割当AI社員数
- タスク進捗率
- 本日の作業内容要約
- 売上または成果指標
- ステータス（稼働中 / 学習中 / 準備中）

例：

```text
営業部
AI社員：5名
進捗：72%
作業：テストコール改善中
売上：¥820,000
```

```text
出版部
AI社員：3名
進捗：45%
作業：ライトノベル校正中
売上：¥120,000
```

必要に応じて、部門サマリー用の型またはMockデータを追加してください。

例：

- CompanySummary
- DepartmentSummary
- DepartmentTaskProgress
- DepartmentRevenueMetric
- AvatarDisplaySettings

## 安全ルール

Claude Codeは以下を行わないこと。

- 実電話番号への発信
- AutoCall本番実行の有効化
- FileMaker本番DBへの書き込み
- Zoom Phone本番APIへの接続
- 実認証情報の保存
- ユーザー操作の無断監視
- secretsの表示
- force push

売上・社員数・進捗はβ版ではMock表示で構いません。実データ連携は後続フェーズです。
