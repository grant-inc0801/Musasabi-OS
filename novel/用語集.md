# 用語集 ―― 経営・起業のことば

> 本編に出てくるビジネス用語・横文字を、**経営者/起業家の目線**でかみ砕いた早見表です。
> 各章末の「**柏木のひとくち経営講座**」と合わせて読むと、物語がそのまま実戦の教科書になります。
> （※初学者向けに噛み砕いた説明です。正確な学術定義とは細部が異なる場合があります。）

---

## 資本・お金まわり

| 用語 | ざっくり説明 | 起業家に刺さる一言 |
| --- | --- | --- |
| **資本／元手（もとで）** | 事業を始めるための最初のお金・資源。 | 多いほど有利。でも「多い＝勝ち」ではない。使い方で決まる。 |
| **キャッシュ / ランウェイ** | 手元の現金と、それが尽きるまでの残り時間（＝滑走路）。 | 黒字でも現金が尽きれば退場。**キャッシュ・イズ・キング**。 |
| **固定費** | 売上に関係なく毎月出ていくお金（家賃・人件費など）。 | 固定費は“重り”。増やす前に「本当に要るか」を三回疑え。 |
| **損益分岐点（そんえきぶんきてん）** | 売上がここを超えて初めて黒字になるライン。 | 「あと何個売れば固定費を回収できるか」を常に暗算しておく。 |
| **ROI／資本効率** | 投じたお金に対して、どれだけ増やせたかの倍率。 | 額（いくら儲けたか）ではなく**率**で見ると、少ない元手でも勝てる。 |
| **利益率** | 売上のうち、手元に残る利益の割合。 | 売上より利益率。安売りは利益率を溶かす“自傷行為”。 |

## 事業モデル・戦い方

| 用語 | ざっくり説明 | 起業家に刺さる一言 |
| --- | --- | --- |
| **プラットフォーム／マッチング（両面市場）** | 「頼みたい人」と「やれる人」など、二種類の利用者を繋ぐ“場”を提供する商売。 | 自分は在庫を持たず、繋ぐだけで手数料が入る。うまく回ると強い。 |
| **テイクレート** | プラットフォームが取引額から取る手数料の割合。 | 高すぎれば逃げられ、低すぎれば儲からない。“ちょうどいい”の設計が肝。 |
| **アセットライト** | 在庫や設備を極力持たない、身軽な事業のかたち。 | 持たざる者の戦法。売れ残り（在庫リスク）で死なない。 |
| **顧客課題／ペイン** | お客が本当に困っていること。 | 「売り込む」前に「聞く」。ペインを当てた者が信用を得る。 |
| **PMF（プロダクト・マーケット・フィット）** | 商品が市場に本当に刺さっている状態。 | “欲しがられて回り出す”瞬間。ここまでは我慢の探索。 |
| **ロングテール** | 一つ一つは地味だが、数を集めると大きい“裾野”の需要。 | 大手が面倒がって拾わない裾野に、持たざる者の勝ち筋がある。 |

## 競争・防御（ここが物語の核心）

| 用語 | ざっくり説明 | 起業家に刺さる一言 |
| --- | --- | --- |
| **価格競争／安売り合戦** | 値段の下げ合いで客を奪い合うこと。 | 資本のある側が必ず勝つ消耗戦。**乗った時点で負け**。 |
| **コモディティ化** | どこで買っても同じ、と思われて値段でしか選ばれなくなること。 | そうなる前に「あなたから買う理由」を作れ。 |
| **参入障壁／モート（濠・堀）** | 後発の競合が簡単には入ってこれない“堀”。 | 値段は一晩で真似できる。**信用・仕組み・関係は真似できない**。 |
| **スイッチングコスト** | 別のサービスに乗り換える手間・損。 | これが高いほど客は離れない。地味だが最強の鎖。 |
| **ネットワーク効果** | 使う人が増えるほど価値が上がる仕組み。 | 評価・実績が貯まるほど、後発は追いつけなくなる。 |
| **レビュー／評価経済** | 実績や評価が可視化され、信用そのものが資産になること。 | 「あそこに頼めば確実」——貯まった信用は、コピーできない濠。 |

## 顧客・成長

| 用語 | ざっくり説明 | 起業家に刺さる一言 |
| --- | --- | --- |
| **ユニットエコノミクス** | 「商品1個・客1人あたり」で採算が合っているか。 | 全体が黒字でも“1個あたり”が赤字なら、売るほど死ぬ。 |
| **LTV（顧客生涯価値）** | 一人の客が、生涯にわたって落としてくれる総額。 | 派手な一発より「また来てくれるか（リピート）」を見よ。 |
| **リピート／常連** | 繰り返し買ってくれる客。 | 新規獲得はコストが高い。常連は利益の源泉。 |
| **一過性需要（ブーム）／ストック需要** | 流行りで一時的に湧く需要／毎日繰り返し発生する需要。 | ブームは去る。豆腐は毎日要る。どちらに賭けるかで戦略が変わる。 |

## 組織・M&A（第2巻で本格化）

| 用語 | ざっくり説明 | 起業家に刺さる一言 |
| --- | --- | --- |
| **経営資源** | ヒト・モノ・カネ・情報。事業を動かす“弾”。 | 何が足りて、何が足りないか。まずは自分の手札を数えろ。 |
| **株式／過半数** | 会社の所有権の分割。過半数を握った者が経営権を持つ。 | 誰に株を渡すかは、心臓を誰に預けるか、と同じ。 |
| **M&A／買収** | 会社を買って、事業や資産ごと取り込むこと。 | 育てるだけが経営ではない。買う・買われるも経営。 |
| **デューデリジェンス（DD）** | 相手や取引の中身を、事前に精査すること。 | “うますぎる話”ほど、契約書の穴を見ろ。 |

---

*本作の主題*：**「価値を奪う経営」と「価値を生む経営」**。
用語はすべて道具にすぎません。道具の使い方に、その人の“経営者としての性根”が出る——というのが、
灰谷湊の物語が問いかけていることです。
