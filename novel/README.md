# 灰からはじめる経営学園
### ―最下層クラスからの下剋上―

> シャッターの下りた商店街で、潰れていく実家の店を見て育った貧乏特待生・灰谷湊(はいたに みなと)。
> 経営者を育てるエリート校「星霜(せいそう)経営学園」で、彼が配属されたのは最下層のFクラスだった。
> 元手も、後ろ盾も、才能の保証もない。あるのは現場で叩き込まれた商売勘と、
> 「なぜ人の店は潰れるのか」を知りたいという執念だけ。
>
> 笑いあり、涙あり、そして――ひと夏の成長あり。
> 星(スター)を賭けて競い合う経営バトルの果てに、灰の中から成り上がる青春ビジネススクール群像劇、開幕。

## 世界観 ―― 星霜経営学園

日本のどこかにある全寮制のエリート校。ここでは座学だけでなく、生徒一人ひとりが
学園通貨「**スター(★)**」を元手に**カンパニー(疑似会社)**を組成し、学園内外の
実ビジネスで利益を競う。

- 半期ごとに**資産と実績でクラス(S / A / B / C / D / F)が再編**される。
- 上位クラスは広い個室寮・潤沢な追加資本・OBネットワークへのアクセス権を得る。
- 下位クラスは相部屋、食堂の割引停止、追加融資の制限――**貧すれば鈍する**を体で学ぶ。
- ゆえに学園では格差がむき出しだ。金を持つ者はさらに稼ぎ、持たざる者は這い上がる術を探す。

「経営とは、限られた資源で最大の価値を生む技術である」――学園長のこの言葉が、
校門をくぐるすべての生徒に突きつけられる最初の試練になる。

## 登場人物

- **灰谷 湊(はいたに みなと)** ――主人公。特待生。現場の商売勘と粘りは誰にも負けない。口癖は
  「値段の裏には、必ず誰かの都合がある」。
  *原体験*：大型スーパーに客を奪われ、潰れていく実家の乾物屋を間近で見た。*苦悩*：「なぜ、いい物を
  作る店が潰れるのか」――その答えを掴むまで、止まれない。
- **白鷺 令子(しらさぎ れいこ)** ――首席入学の財閥令嬢。冷徹で完璧なSクラスの女王。ライバルにして
  ヒロイン。第1巻では、夏の商店街でお忍びの一面や、閉じた店の前で見せる素顔が垣間見える。
  *苦悩*：「愛したものは失う」――だから何も愛さないと決めた、硝子の檻の中の少女。その原体験は第2巻で詳らかに。
- **番場 剛(ばんば ごう)** ――湊と同じ貧乏組。図体はでかいが気は優しい、人たらしの営業型。ムードメーカー。
  *原体験*：大口取引先に切り捨てられ、父の鉄工所が倒産。六人兄弟の長男として働いて弟妹を養った。
  *苦悩*：金なき者に残された唯一の元手は「人に信じてもらう力」だと信じている。だが、それは諸刃の剣でもある。
- **桃園 ひな(ももぞの ひな)** ――数字とデータに異常な嗅覚を持つ小柄なマーケの奇才。よく喋る、よく食べる。
  *原体験*：能力を「気味が悪い」と疎まれ、才能だけ利用されて捨てられてきた。
  *苦悩*：「人は裏切る、データは裏切らない」――人を信じられない。自分もいつか道具として捨てられると怯えている。
- **財前 康介(ざいぜん こうすけ)** ――弁も立ち、見栄えもいい同級生。人の心を読み、動かす才に長ける。
  第1巻では、爽やかな好人物として湊たちの前に現れる。だが、その笑みの奥には――。
  *なぜ彼が、ここまで「奪うこと」に執着するのか。その原体験は、まだ誰も知らない（第2巻にて）。*
  この先、幾度も湊を追い詰めることになる宿敵。
- **黒崎 遼(くろさき りょう)** ――皮肉屋の上級生。かつて頂点に立ちながら、信じた仲間に事業を奪われ転落した
  「伝説の落伍者」。*苦悩*：もう一度火を灯す理由を、失くしてしまった男。湊に世界のルールを教える師匠格。
- **柏木(かしわぎ)先生** ――Fクラス担任の経営学講師。いつも眠そうでネクタイは緩みっぱなし。だが毎朝の小テスト
  『今朝の一問』だけは欠かさない。「出された問いを疑える奴だけが生き残る」が持論。
